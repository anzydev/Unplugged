CORE POWERS

Engineered for absolute reliability and zero background resource footprint.

⚡

Event-Driven

Uses macOS IOKit hardware notification triggers. Runs on absolute zero CPU cycles while idling, firing only when the charger is connected or disconnected.

⏳

Smart Countdown

Configurable safety timers (10s, 30s, 1m, 5m). Instantly aborts the shutdown sequence and restores normal operation if the power cable is plugged back in.

🔔

Rich Notifications

Communicates with the macOS User Notification Center to display power events, updates on remaining time, and alerts before shutdown is triggered.

⚙️

Launch at Login

Utilizes the modern macOS 13+ ServiceManagement API (`SMAppService`) to configure automatic startup, seamlessly matching standard macOS system settings.

🔒

Secure & Transparent

Requires no sandbox but contains zero network connectivity, tracking SDKs, or data uploads. Pure AppleScript and system calls for absolute confidentiality.

📦

Universal Binary

Optimized native build for Apple Silicon (M1/M2/M3/M4) and Intel architectures, compiled using modern Swift 5.9 with strict concurrency checks.

HOW IT WORKS

Three steps to continuous Mac power automation protection.

01

Run the Script

Double-click the downloaded DMG, drag the application into your Applications folder, and launch the utility.

02

Set Your Timer

Click the menu bar icon, open Settings, and pick a custom countdown duration (e.g. 30 seconds). Enable Launch at Login.

03

Automate

Unplug the charger. Watch the countdown trigger in your menu bar. Reconnecting immediately cancels the countdown.
